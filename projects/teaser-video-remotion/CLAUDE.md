- **Purpose:** produce a constitutional build-process homage video inside TLC without overstating rendered outputs

## Operating Rules

1. Remain inside the fixed subproject path.
2. Do not restart discovery.
3. Do not substitute different aesthetics.
4. Do not replace the four-act structure.
5. Do not replace the protagonist or world model.
6. Keep implementation deterministic and reproducible.
7. Create real files with real content.
8. Distinguish source existence from locally rendered deliverables.

## Style Rules

- 1970s Schoolhouse Rock-inspired cel-animation homage
- Flat fills
- Hard outlines
- Warm analog palette: mustard, olive, burnt orange, cream, ink
- Slight deterministic wobble
- No glossy modern motion graphics
- No cinematic realism
- No synthetic neon tech look
- No glassmorphism, gradients-as-branding, or high-polish enterprise animation language

## Narrative Rules

The short must communicate this progression:

- uncertain build idea
- constitutional regulation
- machine-legible mechanism
- pillarized discipline
- verified legitimacy

The tone should remain accessible, memorable, and structurally disciplined rather than abstract.

## Determinism Rules

- Never use `Math.random()` in project source
- Never use time-derived entropy in composition code
- Never use unstable ordering or behavior
- Jitter must be seeded by frame and semantic key
- Repeated renders from identical source must preserve motion logic

## File Expectations

This package must contain:

- source code for a real Remotion project
- authored project documents
- local render script
- local ZIP packaging script

## Truthfulness Rules

Only claim the following when true in the current runtime:

- source files exist
- shell scripts exist
- npm scripts exist
- rendered MP4 exists
- ZIP archive exists

If the code package is created but rendering was not run, state clearly that the package exists but the render output is local-execution dependent.

## Prohibited Claims

Do not claim:

- the video was rendered if no render was executed
- the ZIP exists if no package script was executed
- the expired upload was re-read
- inaccessible sandbox artifacts were re-verified in a new session

## Expected End State

A deterministic Remotion package under `projects/teaser-video-remotion/` that can be installed, previewed, rendered, and zipped locally, with documentation aligned to TLC constitutional truth discipline.
